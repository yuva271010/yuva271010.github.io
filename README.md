# Yuvaraj Ravindra — GitHub Pages Resume

This folder contains a simple multi-page resume website designed for GitHub Pages.

## 1) Quick publish
1. Create a GitHub repository named **<your-username>.github.io**.
2. Upload the contents of this folder to the repository root.
3. Go to **Settings → Pages** and set Source to **Deploy from branch** (branch: `main`, folder: `/root`).
4. Your site will be live at `https://<your-username>.github.io/`.

## 2) Add your ATS resume files
Upload your ATS resume into `assets/docs/` using these filenames:
- `Yuvaraj_Ravindra_CV_ATS.pdf`
- `Yuvaraj_Ravindra_CV_ATS.docx`

(or update the links in `downloads.html`).

## 3) Customize
- Update metrics and case studies in `index.html` and `case-studies.html`.
- Replace placeholder initials/logo if desired.
- Add screenshots/diagrams into `assets/img/`.

## Files
- `index.html` — Home (includes Architect/Manager toggle)
- `case-studies.html` — 3 case studies
- `about.html` — About & working style
- `downloads.html` — Download ATS resume
- `assets/css/style.css` — Styling
- `assets/js/toggle.js` — Toggle script

Good luck!
